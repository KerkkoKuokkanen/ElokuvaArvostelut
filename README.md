# ElokuvaArvostelut

- Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
- Käyttäjä pystyy lisäämään arvostelun jollekkin elokuvalle ja antamaan sille arvosanan
- Käyttäjä näkee sovellukseen lisätyt muiden arvostelut.
- Käyttäjä pystyy etsimään elokuvia hakusanalla ja näkee sille annetut arvostelut.
- Sovelluksessa on käyttäjäsivut, jotka näyttävät käyttäjän lisäämät arvostelut.
- Käyttäjä pystyy valitsemaan lisäksi valitsemaan arvostelulleen luokittelun (suosittelen katsomaa / en suosittele katsomaan).
- Käyttäjä pystyy jättämään palautteen toisen arvostelusta merkkaamalla sen hyödylliseksi.
- Elokuvat näyttävät tilastoja, arvostelujen määrä ja arviointien keskiarvo
